1. PROJECT OVERVIEW

Aljazeera Transits is a logistics brokerage platform that connects customers needing cargo transportation with verified transporters. The platform acts as a middleman, handling bookings, payments, tracking, and commission management.

Business Model:

Customer requests transportation.
Transporters bid or accept jobs.
Customer pays through Aljazeera Transits.
Platform automatically deducts commission.
Remaining balance is paid to transporter after delivery confirmation.

Target Markets:

Tanzania
East Africa
Africa-wide
International Road Freight Corridors

PostgreSQL Schema
CREATE TABLE users (
    id UUID PRIMARY KEY,
    full_name VARCHAR(255),
    email VARCHAR(255) UNIQUE,
    phone VARCHAR(50),
    password_hash TEXT,
    role VARCHAR(20),
    created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE transporters (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    verification_status VARCHAR(20)
);

CREATE TABLE vehicles (
    id UUID PRIMARY KEY,
    transporter_id UUID REFERENCES transporters(id),
    plate_number VARCHAR(50),
    truck_type VARCHAR(100),
    capacity DECIMAL
);

CREATE TABLE shipments (
    id UUID PRIMARY KEY,
    customer_id UUID REFERENCES users(id),
    transporter_id UUID REFERENCES transporters(id),
    pickup_location TEXT,
    destination TEXT,
    cargo_type VARCHAR(100),
    weight DECIMAL,
    status VARCHAR(50)
);

CREATE TABLE payments (
    id UUID PRIMARY KEY,
    shipment_id UUID REFERENCES shipments(id),
    amount DECIMAL,
    commission DECIMAL,
    transporter_share DECIMAL
);
